文本编写使用的是markdown格式的文档，我现在正在学习它的语法。

hexo的几个常见命令：

hexo new ：简写为hexo n，用于创建新的md文档
文章可以拥有如下属性：
- Setting Description Default
1 layout Layout post或page
2 title 文章的标题
3 date 创建日期 文件的创建日期
4 updated 修改日期 文件的修改日期
5 comments 是否开启评论 true
6 tags 标签
7 categories 分类
8 permalink url中的名字 文件名

hexo clean：清除缓存文件 db.json 和已生成的静态文件 public

hexo g：生成网站静态文件到默认设置的 public 文件夹

hexo server：简写为hexo s，用于启动本地服务器
