我的第一篇测试用博客

发表于 2023-10-08 更新于 2023-10-11 Valine：

这是我的第一篇测试性博客，部署在GitHub上，初始地址为huanyi233.github.io

参考的教程：https://zhuanlan.zhihu.com/p/102592286
markdown语法：https://blog.csdn.net/zgdwxp/article/details/103156841
hexo常用的操作：https://www.jianshu.com/p/a2d298e26dcd

我不知道我后续是否会购买域名因为国内购买域名还需要备案挺麻烦的
由于是部署在GitHub上，所以打开时会有一定的延迟和卡顿，并且我现在不能直接在网页上进行增删改，这我认为是及其不方便的。

文本编写使用的是markdown格式的文档，我现在正在学习它的语法。

hexo的几个常见命令：

hexo new ：简写为hexo n，用于创建新的md文档
文章可以拥有如下属性：
- Setting Description Default
1 layout Layout post或page
2 title 文章的标题
3 date 创建日期 文件的创建日期
4 updated 修改日期 文件的修改日期
5 comments 是否开启评论 true
6 tags 标签
7 categories 分类
8 permalink url中的名字 文件名

hexo clean：清除缓存文件 db.json 和已生成的静态文件 public

hexo g：生成网站静态文件到默认设置的 public 文件夹

hexo server：简写为hexo s，用于启动本地服务器
